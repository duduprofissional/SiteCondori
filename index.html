Programação Linear

Entenda os conceitos fundamentais dessa poderosa ferramenta matemática

O que é Programação Linear?

Programação Linear é uma técnica de otimização matemática utilizada para encontrar a melhor solução possível para problemas que envolvem restrições lineares. É amplamente aplicada em áreas como economia, engenharia, logística e muito mais.

Introdução e História

A Programação Linear (PL) surgiu no início do século XX como uma ferramenta para otimização em problemas reais. Foi formalizada por George Dantzig na década de 1940, com o desenvolvimento do Método Simplex, durante a Segunda Guerra Mundial para otimizar recursos militares.

Componentes da Programação Linear

Para formular um problema de Programação Linear, você precisa de:

  • Função Objetivo: A equação que você quer maximizar ou minimizar (ex.: lucro, custo, tempo).
  • Variáveis de Decisão: Representam as escolhas possíveis (ex.: quantidade de itens a produzir).
  • Restrições: Limites matemáticos expressos por desigualdades (ex.: disponibilidade de recursos).
  • Não-Negatividade: As variáveis de decisão geralmente não podem ser negativas.

Exemplo:

Maximizar: Z = 3x + 2y
Sujeito a:
1. x + y ≤ 4
2. 2x + y ≤ 5
3. x, y ≥ 0

Métodos de Resolução

Os problemas de Programação Linear podem ser resolvidos por diferentes métodos, como:

  • Método Gráfico: Ideal para dois ou três parâmetros, pois permite a visualização direta das soluções.
  • Método Simplex: Um algoritmo iterativo usado para resolver problemas mais complexos.
  • Softwares e Ferramentas: Solvers como Excel Solver, Python (bibliotecas como PuLP e SciPy), MATLAB e R podem automatizar a resolução.

Aplicações Reais

Logística: Minimizar custos de transporte.

Finanças: Maximizar retorno de investimentos.

Produção: Otimizar a alocação de recursos para maximizar a produção.

Bibliotecas Python para Programação Linear

PuLP: É uma biblioteca Python usada para modelar e resolver problemas de otimização linear.

Exemplo de código:

from pulp import LpMaximize, LpProblem, LpVariable

# Criar o problema
model = LpProblem(name="programacao-linear", sense=LpMaximize)

# Criar variáveis de decisão
x = LpVariable(name="x", lowBound=0)
y = LpVariable(name="y", lowBound=0)

# Adicionar a função objetivo
model += 3 * x + 2 * y, "Funcao_Objetivo"

# Adicionar restrições
model += (x + y <= 4, "Restricao_1")
model += (2 * x + y <= 5, "Restricao_2")

# Resolver o problema
status = model.solve()

# Exibir resultados
print(f"Status: {model.status}, Solução: x={x.value()}, y={y.value()}")
